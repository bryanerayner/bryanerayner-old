Animated Slideshow boost your theme through shortcode with a beautiful CSS3 animated image and content slideshow.

==Description==

This is a simple slideshow plugin that boost your theme with a beautiful css3 animated slideshow powered with bxslider.

==Frequently Asked Questions==

= What is this plugin and why do I need it? =

This plugin help you create beautiful Slideshow with CSS3 Animation via Custom Post Type

= How to Use this? =

This plugin runs with shortcode use [as-slider] or [as-slider ids=""] to run the shortcode.

==Installation==

This plugin can be installed directly from your site.

1. Log in and navigate to Plugins &rarr; Add New.
2. Type "Animate Slider" into the Search input and click the "Search" button.
3. Locate the Animate Slider in the list of search results and click "Install Now".
4. Click the "Activate Plugin" link at the bottom of the install screen.

It can also be installed manually.

1. Download the "Animate Slider" plugin from WordPress.org.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the "Plugins" screen.
4. Locate "Animate Slider" in the list and click the "Activate" link.
